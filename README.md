# SkillBuild Claude Marketplace

A community library of AI skills for Claude, built with [SkillBuild.ai](https://skillbuild.ai).

## Structure

Skills are organised by audience:
- `skills/internal/` — Internal team workflows
- `skills/clients/` — Client-facing skills
- `skills/executives/` — Executive reporting skills
- `skills/general/` — General purpose skills

Each skill is a `SKILL.md` file ready to upload to Claude.
